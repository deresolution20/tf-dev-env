# tf-dev-env
Building a remote TF dev environment in AWS
 

<h2>Description</h2>
Automate building a dev env with docker installed and ssh via vscode.  


<br />


<h2>Languages and Utilities Used</h2>

- <b>Bash</b> 
- <b>Terraform </b>

<h2>Environments Used </h2>

- <b>Win 11</b>
- <b>Ubuntu 18 server</b>
